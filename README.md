# sorting visualizer
## used SDL libary for vis
command to run -> g++ -o program platformer.cpp -I/usr/include/SDL2 -lSDL2 
Program generates random nums from 1-99 and sorts them using 
1) Bubble Sort
2) Selection
3) Insertion
